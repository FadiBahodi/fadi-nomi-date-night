# Fadi & Nomi — Date Night Atlas, London Ontario

A working atlas of date nights in and around London, Ontario, built from Old North (home base).

## Pages

- `index.html` — landing page with all 11 itineraries
- `today.html` — Saturday April 25, 2026 hour-by-hour with Plan A / Plan B / Plan C (puppy-along)
- `itineraries.html` — 11 complete date-night itineraries with timelines and addresses
- `guide.html` — deep-dive reference: restaurants, bars, activities, day-trips, hidden gems
- `map.html` — interactive Leaflet map with all 60+ stops, category filters

## Built From
- Tourism London, Reddit `/r/londonontario`, OpenTable, blogTO, Tripadvisor, Yelp
- Local blogs and Western Gazette restaurant coverage
- Verified addresses, phone numbers, and Saturday April 25, 2026 hours where critical
- Closure list (don't waste a drive on Honest Lawyer, Norma Jean's, etc.)
